# Vehicle Fleet Management
Desktop software to manage vehicle fleets (maintenance, fuel entries, etc)
## Developed on Netbeans with Java Swing
### Functionalities:
* Cars
* Employees
* Suppliers
* Fuel entries
* Maintenances
* Accidents
* Tickets
* Deliveries
* Backup records on .csv files
